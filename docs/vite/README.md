## 快速开始

### 创建一个简单的ts项目

:::code-group

```shell [pnpm]
pnpm create vite
```

```shell [yarn]
yarn create vite
```

```shell [npm]
npm create vite@latest
```
:::

使用任意其中一个工具，执行命令，根据提示输入项目名，选择`framework`选择`Vanilla`, `variant` 选择`TypeScript`
